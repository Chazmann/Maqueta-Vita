• Sobre cada turno se podra:  ( Ver flujo y reglas de atencón al final del documento – Punto 1,2,3)
    • Ver la historia clínica del Paciente
    • Iniciar la Atención
    • Llamar paciente ( Eventual integración con llamador )
  1. Boton “Iniciar Atención”
    1. Al seleccionar “Iniciar Atención” desde la sala de espera, el sistema debera chequear que el profesional no haya alcanzado el límite de borradores.
    2. En caso de que el profesional se encuentre en el límite de borradores. Se debera desplegar una alerta que anuncie:
      1. Has alcanzado el límite de borradores de Evolución en curso ( 5 ), para iniciar la atención de un paciente debe finalizar un borrador en curso. 
    3. En caso de que NO se encuentre en el límite de borradores, el sistema despliega la HC del paciente y muestra el menú de selección de evolución. 
  2. Finalizar atención
    1. Durante la atención de un turno, el usuario debe visualizar un boton que le permita Finalizar la atención. Lo que cambiara el estado del turno a “Atendido” y llevara al usuario a la vista de sala de espera. 
    2. En caso de que la evolución en curso no se encuentre firmada, el sistema desplegara un modal que anuncia: 
      1. La evolución actual no se encuentra firmada.
        1. El modal contara con 3 opciones: 
          1. Cancelar: Cierra el popup
          2. Enviar a borradores : Muestra sala de espera, Evolución queda en borrador
          3. Firmar : Firma evolución en curos, muestra sala de espera
  3. Ver Historia Clínica
    1. Permite acceder a la historia y visualizara, no inicia la atención. No permite redactar evolución ni generar órdenes. En caso de intentar realizar una de estas acciones debe desplegar un error “Debe iniciar la atención para realizar esta acción”
